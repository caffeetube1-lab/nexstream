# NexStream - Universal Video Downloader

NexStream is a premium, high-performance web application designed for downloading videos from any platform (YouTube, TikTok, Instagram, etc.) with maximum quality and speed.

## Features
- **Universal Support**: Powered by `yt-dlp`, works with hundreds of sites.
- **Perfect Quality**: Automatically fetches the highest available resolution.
- **Sleek UI**: Modern glassmorphism design with smooth animations.
- **Fast Extraction**: Instant video metadata retrieval.
- **Audio Extraction**: Option to download as high-quality MP3.

## Setup Instructions
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Ensure you have [yt-dlp](https://github.com/yt-dlp/yt-dlp) installed or in the project folder.
3. Open your terminal in this directory.
4. Run `npm install` to install dependencies.
5. Start the server:
   ```bash
   node server.js
   ```
6. Open your browser and navigate to `http://localhost:3000`.
